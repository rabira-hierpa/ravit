# RaViT - Rails, Vite and Tailwindcss

This is a template for a Rails 6.1 application with Vite and Tailwindcss.

## Installation

```bash
git clone https://github.com/rabira-hierpa/ravit.git
cd ravit
bundle install
yarn install
```

## Running the app

```bash
rails s
```
